What is an Armstrong Number?

An Armstrong number (also known as a narcissistic number) is a number that is equal to the sum of its own digits each raised to the power of the number of digits. For example:

153 = 1³ + 5³ + 3³ = 1 + 125 + 27 = 153

How to Check for an Armstrong Number?

To determine if a number is an Armstrong number:

  • Find the number of digits (n) in the number.
  • Raise each digit to the power of n and sum the results.
  • If the sum equals the original number, it is an Armstrong number.

Here's an example for the number 9474:

  • 9474 = 9⁴ + 4⁴ + 7⁴ + 4⁴ = 6561 + 256 + 2401 + 256 = 9474

How Armstrong Numbers are Used in Code

In programming, checking for Armstrong numbers often involves the following steps:

1. *Input*: A user provides a number (n).
2. *Digit Extraction*: Break the number into individual digits.
3. *Power Calculation*: Raise each digit to the power of the number of digits and calculate their sum.
4. *Comparison*: Compare the calculated sum to the original number.

Armstrong numbers are interesting in mathematical puzzles and coding exercises for beginners.
