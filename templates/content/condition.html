What Are Conditional Statements?

Conditional statements allow us to make decisions in our programs. They let the program decide which action to take based on whether a condition is true or false.

In most programming languages, the basic structure of a conditional statement is:

if (condition) {
    // Code to run if the condition is true
} else {
    // Code to run if the condition is false
}

Now, let's see how this works in different languages!

C Programming

C allows us to use conditional statements like if-else to control the flow of our program. Here's an example:

#include <stdio.h>

int main() {
    int age = 18;
    if (age >= 18) {
        printf("You are an adult.\n");
    } else {
        printf("You are a minor.\n");
    }
    return 0;
}

C++ Programming

C++ extends C and supports object-oriented programming. Here's an example of an if-else statement:

#include <iostream>

using namespace std;

int main() {
    int age = 18;
    if (age >= 18) {
        cout << "You are an adult." << endl;
    } else {
        cout << "You are a minor." << endl;
    }
    return 0;
}

Java Programming

Java uses if-else statements for decision-making. Here's an example:

public class Main {
    public static void main(String[] args) {
        int age = 18;
        if (age >= 18) {
            System.out.println("You are an adult.");
        } else {
            System.out.println("You are a minor.");
        }
    }
}

Python Programming

Python uses simple and readable syntax. Here's an example of an if-else statement:

age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

JavaScript Programming

In JavaScript, if-else statements are used similarly. Here's an example:

let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}
