What is a Variable?

A variable is like a box that can hold information. You can name the box and store things in it, like numbers, words, or even more complex data. Think of it as a box where you can put information, label the box, and use it whenever needed.

example

Why Use Variables?

Variables make it easy to store and reuse information in your program. Instead of repeating values, you can use a variable to store them once and refer to them whenever needed.

Types of Variables

Variables can store different types of data, depending on the programming language:

  • Integer: Stores whole numbers (e.g., 42).
  • Float/Double: Stores decimal numbers (e.g., 3.14).
  • String: Stores text (e.g., "Hello, World!").
  • Boolean: Stores true or false values (e.g., true, false).
  • Others: Includes arrays, objects, etc., depending on the language.

Rules for Naming Variables

  • Variable names must start with a letter, an underscore _, or a dollar sign $ (in JavaScript).
  • Variable names cannot contain spaces.
  • Variable names are case-sensitive, so age and Age are different.
