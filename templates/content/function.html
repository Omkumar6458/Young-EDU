What is a Function?

A function is a block of reusable code that performs a specific task. Functions allow you to organize your code into smaller, manageable chunks. You can call a function whenever you need it, making your code more efficient and easier to read.

Functions in C

In C, a function is declared with a return type, function name, and parameters (optional). The function performs a task and can return a value.

Example:

#include 

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 4);
    printf("The sum is: %d", result);
    return 0;
}

Functions in C++

C++ functions are similar to C but allow for more advanced features like function overloading. Functions in C++ are declared similarly to C functions.

Example:

#include 
using namespace std;

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 4);
    cout << "The sum is: " << result << endl;
    return 0;
}

Functions in JavaScript

In JavaScript, functions can be declared in various ways, including function expressions, arrow functions, and function declarations.

Example:

function add(a, b) {
    return a + b;
}

let result = add(3, 4);
console.log("The sum is: " + result);

Functions in Python

Python functions are simple to declare and don't require a return type. Functions in Python can return values, take parameters, and have default values for arguments.

Example:

def add(a, b):
    return a + b

result = add(3, 4)
print("The sum is:", result)
